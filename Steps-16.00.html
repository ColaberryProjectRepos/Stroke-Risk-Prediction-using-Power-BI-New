Creating Measures

There are several ways to create measures.

Click on Enter Data from the home tab to create a table, name it “Measures Table” and click on load. A new table named Measures Table appears on the Data Pane. This table will contain all the measures we need for this report.


Right click on the Measures table and click on New Measure or From the Home Tab, either on Data View or Report View, Click on New Measure from the ribbon. All the options will land on the same spot. On the Formula bar, copy and paste the DAX formula for the measure. Once done press Enter or Click on commit (Checkmark icon) on the left of the formula. After Creating each measure, choose the appropriate format and decimal for each value (Measure) and click on save.

Listed measures along with their DAX formulas

  • Patients = COUNTROWS('strokedata') [Format: Whole Number]
  • Stroke% = IF(ISBLANK(DIVIDE(COUNTROWS(FILTER('strokedata', 'strokedata'[stroke] = 1)), COUNTROWS('strokedata'))), " ", DIVIDE(COUNTROWS(FILTER('strokedata', 'strokedata'[stroke] = 1)), COUNTROWS('strokedata'))) [Format: Percentage]
  • Heart% = IF(ISBLANK(DIVIDE(COUNTROWS(FILTER('strokedata', 'strokedata'[heart_disease] = 1)), COUNTROWS('strokedata'))), " ", DIVIDE(COUNTROWS(FILTER('strokedata', 'strokedata'[heart_disease] = 1)), COUNTROWS('strokedata'))) [Format: Percentage]
  • Hyper% = IF(ISBLANK(DIVIDE(COUNTROWS(FILTER('strokedata', 'strokedata'[hypertension] = 1)), COUNTROWS('strokedata'))), " ", DIVIDE(COUNTROWS(FILTER('strokedata', 'strokedata'[hypertension] = 1)), COUNTROWS('strokedata'))) [Format: Percentage]
