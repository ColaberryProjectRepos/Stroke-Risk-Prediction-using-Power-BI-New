Challenges and Solutions

  1. Dataset Quality: Acquiring a high-quality dataset was a major obstacle, as many datasets had gaps that made them unsuitable for the project. After working with two different datasets, which proved challenging for visualization, I ultimately used a third dataset for the final project.

  2. Code Compatibility: Running Python code generated in Numtra within Microsoft Fabric was a significant challenge. Some code had to be rewritten to ensure compatibility with the Fabric domain.

  3. DAX Formulas and Table Sorting: Understanding and creating DAX formulas and sorting tables was challenging. To address this, I watched YouTube videos and consulted Chat GPT for accurate DAX formulas. Additionally, creating step-by-step GIFs under 5MB was time-consuming, which I mitigated by increasing my clicking speed.


Lessons Learned

Data preparation is a crucial step in the data analytics pipeline, ensuring that the data is of high quality and ready for analysis, which leads to more accurate insights and decisions. Maintaining detailed documentation throughout the project process enhanced transparency and provided valuable reference for future endeavors.
