Importing Data into Power BI

Dataset source Link: 
https://www.kaggle.com/datasets/fedesoriano/stroke-prediction-dataset

Project Essentials File Zip:
It contains important files used for this project. Download zip file and save it to your computer.

Detailed Instructions:

1. Open Power BI Desktop
2.  Click on Get data ribbon.
3. Select Text/CSV and then click connect
4. Open the saved file, choose ‘strokedata’ file and click on Open
5. A preview of the data will be displayed.
