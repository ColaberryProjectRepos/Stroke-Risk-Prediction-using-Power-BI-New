Adding canvas Background, Colaberry Logo, and Title
Adding canvas Background/Watermark
From the report view, go to Visualization pane -> Format-> Canvas Background-> Image Browse -> select Image (backgroundpic) from the folder you downloaded earlier -> click Open 
Image Fit = Fill
Choose Transparency = 55%



Adding Colaberry Logo
On Report View, Go to Insert Tab
Click on Image on the ribbon,
Select the Colaberry Logo from the folder you downloaded earlier and click Open.
Make sure to format the logos according to these specifications: Height = 97, Width = 130, Horizontal = 0, Vertical = 0


Adding Title
   Go to the "Home" tab in the Power BI ribbon and click on the "Text box" icon. This will allow you to insert a text box onto your report canvas.
With the text box selected, you'll see the "Format" tab in the "Visualizations" pane on the right. Then click on General->  properties: Size: Height=97, Width=1147 position: Horizontal=130, vertical =0
Padding: Top =20px
Title ON: Stroke Risk Prediction Overview Dashboard
Heading 3, Font: Segoe UI Bold, Font size = 25, Text color: #0A598A, Center Alignment
Background ON, Color #FFFFFF
