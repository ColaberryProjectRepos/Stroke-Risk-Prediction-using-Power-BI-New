Creating a relationship between the main table and the sorting tables

1. Create the Sorting Table
Click on Enter Data from the home tab to create a table with 2 columns and name it “Bim Sorted Table”. Rename the columns name by doubling clicking in the name box and enter data into the table. After entering the data click on load.
Follow the steps in the GIF below:


All the tables should look like this after entering data


2. Create the Relationship:
Click on the "Model" icon on the left-hand side of Power BI Desktop to switch to the model view.
In the model view, find your main table and the new sorting table.
Click on the Bmi Group column in the main table and drag it to the Bmi Group column in the sorting table.
This will open the "Create Relationship" dialog box.
Ensure the relationship is many-to-one, with your main table on the "many" side and the sorting table on the "one" side.

Click "Save" to create the relationship.

At the end all the tables with the relationship should look like this:


3.Use the Sort by Column Feature:
Go back to the "Report/ data" view.
Select the sorting table.
Click on the Bmi Group column.
In the ribbon, choose "Sort by Column" and select Bmi Group Order from the sorting table.


We will sort the Age Group and Glucose Level Group column using the same procedure.
